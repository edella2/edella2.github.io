Enumerable Methods!

Enumerable#each_slice(n)

10/10/2015

Before this week, I had always considered Enumerable methods confusing (OK, maybe They're still a little confusing). Ruby has several of these built in that makes searching or going through an array or hash much easier.

This week we had a challenge that involved adding commas to to numbers. We wanted integers like 1000 to return 1,000. This was a tricky challenge that I spent a lot of time on this week working to refactor to be as clean as possible.

My solution for it used an enumerable #each_slice(n):

def separate_comma(num) new_num = num.to_s.reverse.split("") output = [] new_num.each_slice(3) do |word| output << word.join("") end output.join(",").reverse end

The use of #each_slice(3) took an array(new_num) and seperated it into chunks of 3. I could then combine those digits into a 3 character long string and put it into my output array. This way if I entered a number like "1234" into the method. My output array in the #each_slice(3) would be ["123","4"]. This made it possible for my final output to be joined together again with a comma seperating each character set in the array.

This image above from globalnerdy is a good example of what each_slice does. It takes a large array containing things like the baseball, smiley face, spelling block and teddy bear and sorts them into new arrays in groups of 3. They have other examples of other enumerable methods on their site and I recommend looking through them for simplified explanations!
