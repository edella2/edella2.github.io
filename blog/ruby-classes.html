Puppy training

10/17/2015

When you first get a puppy you get to do one of the most exciting things with it right away. You can name it!

class Puppy   def initialize(name)     @name = name   end end

But having a puppy that just has a name isn't exciting. Things that make puppies so much fun is seeing them wag their tail when you say their name!

class Puppy   def initialize(name)     @name = name   end      def wag     print "#{@name} is such a good dog!"     print "*#{@name} wags their tail*"   end end

The above is an example of why you often want to use instance variables instead of local variables in classes. Because our @name is an instance variable @name can be used with any method inside of the puppy class. Instance variables should be used where it make sense, the puppy's name will always be the same throughout the class and may need to be reused for other methods or tricks we want to teach our puppy.
