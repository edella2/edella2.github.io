Big-O notation

and examples of it

10/31/2015

Big-O notation is used to classify the growth rate of a function. In computer science specifically it is used to classify an algorithms processing time or space requirements.

Complexity of the function is expressed with big-O notation. It looks at the highest growth rate of a function to describe it.

  • A linear algorithm (y = 2x + 5) is written O(N)
  • A quadratic algorithm (y = 5x^2 - 2x + 5x +3) is written O(N^2)
  • A final example (y = 4x^5 + 3x^2 - 2x^3 + 4) is written O(N^5)

As you can see the part of a function that can have the most change defines the function.

We can look at other examples using functions we might use in our programs in a different way. If we had a function like:

def big_O_N(array)  array.each { |element| puts element } end

The complexity of this function would be written as a O(N) using big-O notation.
