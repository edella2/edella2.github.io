Integers

#even? // #odd?
2.even? #=> true
#to_i // #floor

Can convert strings with digits as characters to numbers. Or floats to integers

2.0.floor #=> 2 2.0.to_i #=> 2 "2".to_i #=> 2
#to_s

Convert a number to a string

2.to_s #=> "2"

Strings

#to_i

Convert a string containing number to an integer

"2".to_s #=> "2"
string << string

concatenates a string

"hello" << " world" #=> "hello world"
#split(pattern, limit)

If a pattern or limit is ommitted the string will be converted into an array containing each charachter of the string.

"hello".split() #=> ["h","e","l","l","o"]
#strip

removes whitespace from a string

" spaces " #=> "spaces"

Arrays

arr[idx] // arr.at(idx)

Access the value stored in the array at the index. Arrays first index is zero

array = [1,"hello", true] array[1] #=> "hello"
arr[idx] // arr.at(idx)

Access the value stored in the array at the index. Arrays first index is zero

array = [1,"hello", true] array[1] #=> "hello"
#insert(n, thing/things to insert)

inserts things into an array at the index specified

array = [1,"hello", true] array.insert(1, "inserted!") array #=> [1,"inserted!", "hello", true]
#pop

Remove the last item in the array and return it.

array = [1,"hello", true, "popped"] array.pop #=> "popped" array #=> [1,"hello", true]
#delete(element)

Remove the element specified in the array.

array = [1, 1, 1, "hello", true] array.delete(1) array #=> ["hello", true]

Hash

Implementing a hash

different ways to implement a hash

hash = { :rocket => "zoom", :name => "Eric" } hash = { symbol: "not a rocket", name: "Eric" }
Accessing a hash by key

You can access hash values by their key.

hash = { symbol: "not a rocket", name: "Eric" } hash[:name] #=> "Eric"
#hash_key?

Returns true or false if hash contains a key

hash = { symbol: "not a rocket", name: "Eric" } hash.has_key?(:name) #=> True
